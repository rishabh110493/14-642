int syscall_read ( int  file,
char *  ptr,
int  len 
)

See linux man page for read.

Parameters
filefile descriptor to read from. In our kernel, we only support STDIN.
ptrpointer to buffer to read from
lenmax length to read
Returns
-1 on error, otherwise number of bytes read

Reads until all len bytes are read or newline/return is found. Detects backspace and EOL characters and handles them appropriately.

The idea is to mimic normal terminal reading input, that is, if the user hits backspace it should erase the previously read character.

References printk(), uart_get_byte(), and uart_put_byte().
