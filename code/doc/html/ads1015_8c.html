uint16_t adc_read ( uint8_t  channel)

Algorithm: Checks if the channel set in the ADC is the same as the channel which is desired to be read. If YES, read from the Conversion register and return the 16 bit value read. If NO, then write to the pointer register the address of the config register, then write to the config register with the appropriate encoding for the MUX (for the desired channel).
